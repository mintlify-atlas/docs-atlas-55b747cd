      Quick Start
    </h2>
    <p className="text-lg text-gray-600 dark:text-gray-400">
      Get Default Tamer running in minutes
    </p>
  </div>

  <Steps>
    <Step title="Download and Install">
      Download the latest DMG from [GitHub Releases](https://github.com/0xdps/default-tamer/releases), drag Default Tamer to Applications, and launch it.
    </Step>
    <Step title="Set as Default Browser">
      Click **"Open System Settings"** in the first-run window, then go to **System Settings → Desktop & Dock → Default web browser** and select **Default Tamer**.
    </Step>
    <Step title="Create Your First Rule">
      Open Default Tamer from the menu bar and create a routing rule — for example, send all Slack links to Chrome, or route work domains to Safari.
      
      <Accordion title="Example: Route Slack to Chrome">
        1. Click the Default Tamer icon in the menu bar
        2. Select **Rules**
        3. Click **Add Rule**
        4. Choose **Source App** → Select **Slack**
        5. Choose target browser: **Chrome**
        6. Click **Save**
      </Accordion>
    </Step>
    <Step title="Test Your Setup">
      Click any link in the app you configured (e.g., Slack) — it should open in your target browser automatically. Hold **⌥ Option** while clicking to manually choose a browser.
    </Step>
  </Steps>
</div>
